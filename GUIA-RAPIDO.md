# 🚀 Guia Rápido - Sistema de Contabilidade

## Para Começar

### 1️⃣ Primeiro Acesso (Cadastro)
```
1. Abra index.html no navegador
2. Clique em "Cadastre-se aqui"
3. Preencha:
   - Nome: Seu nome completo
   - CPF: Apenas números (11 dígitos)
   - Senha: Mínimo 4 caracteres
   - Confirmar senha
4. Clique em "Cadastrar"
```

### 2️⃣ Login
```
1. Digite seu CPF (só números)
2. Digite sua senha
3. Clique em "Entrar"
```

### 3️⃣ Cadastrar Outros Sócios
```
Cada sócio deve repetir o passo 1 (Cadastro)
```

## Usando o Sistema

### ➕ Adicionar Distribuição de Lucros
```
1. Vá na aba "Distribuição de Lucros"
2. Clique em "➕ Adicionar Registro"
3. Preencha:
   - Data
   - Selecione o sócio no dropdown
   - Descrição (opcional)
   - Valor
   - Observações (opcional)
4. Salva automático!
```

### 📈 Adicionar Rendimentos
```
1. Vá na aba "Rendimentos Financeiros"
2. Clique em "➕ Adicionar Registro"
3. Preencha os campos
4. Salva automático!
```

### 👥 Ver Todos os Sócios
```
Clique no botão "👥 Gerenciar Sócios" no topo
```

### 📥 Exportar para Excel
```
Clique em "📥 Exportar Excel" na aba desejada
O arquivo .csv pode ser aberto diretamente no Excel
```

### 🚪 Sair
```
Clique em "🚪 Sair" no canto superior direito
```

## Atalhos de Teclado

| Atalho | Ação |
|--------|------|
| `Ctrl + 1` | Ir para aba Distribuição de Lucros |
| `Ctrl + 2` | Ir para aba Rendimentos Financeiros |
| `Ctrl + N` | Adicionar novo registro na aba atual |

## ⚠️ Importante

✅ **Cada sócio vê apenas os registros que ELE criou**  
✅ **Todos podem selecionar qualquer sócio nos registros**  
✅ **Dados salvos automaticamente no navegador**  
⚠️ **Faça backup exportando para Excel regularmente**  
⚠️ **Não limpe os dados do navegador ou perderá tudo**

## 💡 Dicas

1. **Senhas simples:** Como é local, use senhas fáceis de lembrar
2. **CPF sem formatação:** Digite apenas os 11 números
3. **Exportação:** Faça backup semanal exportando para Excel
4. **Seleção de sócio:** Use o dropdown, não precisa digitar

## 🆘 Problemas Comuns

**Erro: "CPF não encontrado"**
- Certifique-se de ter se cadastrado primeiro
- Use apenas números no CPF

**Erro: "Senha incorreta"**
- Verifique se digitou corretamente
- Senhas são case-sensitive

**Não vejo meus dados**
- Verifique se está logado com o CPF correto
- Cada usuário tem dados separados

**Perdi meus dados**
- Se limpou o cache do navegador, os dados são perdidos
- Por isso é importante exportar regularmente!
